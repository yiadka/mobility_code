# mobility_code
## How to run
仮想環境に入って
```
python main.py
```

## dependent
`requirements.txt`に使用したライブラリをまとめてます。
私は`venv`を使って仮想環境を切っているので、任意の仮想環境に入って
```
pip install -r requirements.txt
```
としてライブラリを一括してインストールしています。
